# MicroCalcu

  ## 一款实现大学生总评计算汇总的软件

许多大学老师在期末之后对学生的成绩进行汇总计算时总会苦恼于Excel复杂的函数公式，使用MicroCalcu可以轻松搞定对Excel表格文件的计算操作。

下面将介绍MicroCalcu的使用方法。

# 软件安装

在下载完安装包后，直接**双击打开安装包**。

<img src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165838349-550099167.png"/>

打开安装包之后，可以默认勾选创建桌面快捷方式，然后**点击Next**。

<img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421170202842-1097338347.jpg"/>




然后**点击Install**，即可开始安装。

<img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421164836948-1715432880.png"/>


安装完成之后我们发现桌面新增了一个**MicroCalcu图标**，即表示安装成功。

![](https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421164918853-933164452.png)



# 使用

**双击打开MicroCalcu**，下面将介绍软件的使用方法（以西安石油大学-大学英语1为例）。

<img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165351856-1323286757.png"/>



下图是西安石油大学-大学英语1的总评组成，我们将按照此要求设置参数。

![](https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165406268-2117618986.png)



1. 首先平时分占比30%，我们只需在“平时”后面的文本框中输入30，或者点击“\^”形状的按钮，可以以5为单位递增。

2. 我们可以看到平时分的30%中有10%的作业成绩（1次作文，1次翻译，3听写），15%的线上听力成绩和5%的实践成绩。这些项目需要在“项目名称”所在的表格中设置。

   1. **双击**项目1，输入“**作业**”两个字（无需引号），在“项目比例“中双击，输入**0.1**（**输入的是小数**）。

   2. 按照上一步骤，依次设置“线听”和“实践”。

   3. 作业中包含的1次作文，1次翻译，3听写，我们需要点击作业一栏中的**Setting
      more**按钮来设置。在弹出的窗口中，输入以下内容后便设置完成了“平时”部分。

      <img style="width:432px;height:243px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165428899-2127500313.png"/>


      平时部分设置完成后最终界面如下所示。
      
      <img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165444979-1042326441.png"/>



3. 接下来点击期中/实验/实践按钮，设置期中部分。按照所给要求，期中占比10%，我们只需在期中后的文本框输入10即可。**由于期中部分不包含其他子项目，项目名称的表格无需更改。**该部分最终界面如下所示。

   <img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165508837-1642480228.png"/>


4. 按照要求期末部分设置最终界面如下。

   <img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165525017-1506020559.png"/>


5. 设置完所有参数后点击**导入Excel按钮**，选择要导入的文件。然后便可得到下图所示窗口。


   <img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165549605-1834694414.png"/>



   该窗口从上到下依次显示了，相关统计数据，（标准差、平均分、最高分、最低分），作文、翻译等各项平均分柱状图以及优良中差所占比例饼图。最下面是计算完成的结果预览。可点击**导出Excel按钮**，将结果导出（点击到导出按钮时会弹出另存为窗口，设置导出文件名称和位置）。当屏幕右下方弹出**导出成功**提示后即表示成功导出了Excel文件。

   <img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165601981-1181174932.png"/>



   此时我们便可以看到导出的Excel文件，如下所示。

   <img src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421165623284-1737025164.png"/>



# 彩蛋

在软件主界面的最上方我们看到有一个**搜索框**。在搜索框中输入内容后按下回车键，会打开电脑默认浏览器，使用**必应**搜索你所输入的内容。

<img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421170523072-401590287.jpg"/>




此外，在搜索框的右边我们设置了问候模块，软件会或去计算机用户名称，并根据系统时间来问候。

<img style="width:600px;height:450px" src="https://img2020.cnblogs.com/blog/2305515/202104/2305515-20210421170533999-1756665979.jpg"/>
